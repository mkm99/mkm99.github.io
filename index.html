About Me

Hi! My name is Mary Machado. I graduated with honors from Rutgers University in May 2020 and earned a Bachelor of Arts in Computer Science with a minor in Mathematics.

I have participated in two internships. In the first internship I worked in a team researching the adsorption properties of a material called “Biochar" and how much of a contaminant would be adsorbed by it. In order to see the changes overtime, we used a special machine to make readings every hour for 3-4 days. Big amounts of data was created, which were analyzed using scripts in Python and created user friendly tables and graphs that showed all the changes. In the second internship, I was responsible to create a server in the cloud using Amazon Web Services. I oversaw the installation, configuration, troubleshoot and deployment of ArcGIS Enterprise, ArcGIS Server and ArcGIS GeoEvent Server.

Both internships taught me the importance of communication between coworkers to accomplish planned objectives. Also, I learned that I can turn out well in a fast pace environment.

Currently, I am learning new technologies and working on new projects, for example I created this website from scratch, and I am currently learning another languages. If technology does not stop innovating, why should I?!?!
